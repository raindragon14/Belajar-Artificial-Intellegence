# Belajar-Artificial-Intellegence

Memahami Artificial Intelligence (AI) memang terasa sulit, namun tujuan utamanya cukup simpel yaitu menciptakan sistem cerdas yang mampu belajar dari pengalaman untuk melakukan penalaran kognitif. 

AI dirancang bukan untuk menggantikan, melainkan untuk membantu kemampuan manusia dalam memecahkan masalah rumit. Keunggulannya adalah kecepatan dan skalabilitas dalam memproses data besar dan memungkinkan kita melihat korelasi serta pola data yang mustahil didapatkan melalui metode biasa.
